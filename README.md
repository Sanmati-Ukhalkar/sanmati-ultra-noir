# Sanmati Ultra Noir Portfolio

A modern, responsive personal portfolio website built with React, Vite, Tailwind CSS, and shadcn-ui. This repository showcases my projects, skills, and professional experience in a sleek "ultra noir" aesthetic.

## Features

- **Hero Section**: An engaging introduction with a stunning dark mode aesthetic.
- **Projects Showcase**: A curated gallery of my latest projects and work.
- **Skills Matrix**: A comprehensive overview of my technical stack and proficiencies.
- **Responsive Design**: fully optimized for desktop, tablet, and mobile devices.

## Tech Stack

This project leverages modern frontend technologies for optimal performance and developer experience:

- **Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn-ui (Radix UI + Tailwind)
- **Routing**: React Router
- **State Management / Data Fetching**: React Query
- **Icons**: Lucide React

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**
   ```sh
   git clone https://github.com/Sanmati-Ukhalkar/sanmati-ultra-noir.git
   cd sanmati-ultra-noir
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

## Deployment

The project is configured for easy deployment. You can build the production assets using:

```sh
npm run build
```

This will generate a `dist` folder containing the optimized production build, ready to be hosted on Vercel, Netlify, GitHub Pages, or any static hosting provider.
